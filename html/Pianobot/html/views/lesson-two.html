Lesson 2

Objective

When large vehicles are backing up they make repeated beeping noise. Explore how the code is written to make the Linkbot beep continuously like a truck backing up.

#!/usr/bin/env python3

# Filename: Lesson02.py
import linkbot
import time

robot = linkbot.Linkbot('')

for i in range (10):
    robot.setBuzzerFrequency(4186.01)
    time.sleep(.5)
    robot.setBuzzerFrequency(0)
    time.sleep(.5)
