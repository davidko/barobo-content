Lesson 5

Objective

The piano keyboard is made up of notes with different frequencies. The Linkbot can play these notes if you enter the frequencies. Letâ€™s try to enter "Do Re Mi Fa Sol La Ti Do", it starts at middle "C" which has a frequency of 1048.5 and goes to "D" with frequency 1174.7, and so on. Notice it takes a new line of code for each note, make the sleep time for each note the same.

#!/usr/bin/env python3

# Filename: Lesson05.py
import linkbot
import time

robot = linkbot.Linkbot('')

robot.setBuzzerFrequency(1046.5)
time.sleep(.5)
robot.setBuzzerFrequency(1174.7)
time.sleep(.5)
robot.setBuzzerFrequency(1318.5)
time.sleep(.5)
robot.setBuzzerFrequency(1396.9)
time.sleep(.5)
robot.setBuzzerFrequency(1568.0)
time.sleep(.5)
robot.setBuzzerFrequency(1760.0)
time.sleep(.5)
robot.setBuzzerFrequency(1975.5)
time.sleep(.5)
robot.setBuzzerFrequency(2093.0)
time.sleep(.5)
robot.setBuzzerFrequency(0)
