Python

Windows

To get Python up and running, we recommend downloading and installing our PortablePython Barobo Edition all-in-one installer at http://my.barobo.com/download/windows_python_allinone_installer.

Once you have downloaded the file, double click on the installer to begin the installation process. You should see a screen similar to the follwing screenshot:

Figure 1.1.  Installing PortablePython - Barobo Edition, Step 1


Click the "Next" button to continue. You should see something similar to the following screenshot:

Figure 1.2.  Installing PortablePython - Barobo Edition, Step 2


At this point, you will need to click the "Browse" button to choose a location to install PortablePython. After clicking on the "Browse" button, you should see the following screen:

Figure 1.3.  Installing PortablePython - Barobo Edition, Step 3


On this screen, you have several options for where to install PortablePython. The easiest place to install it is onto your Desktop. If that is where you would like to install it, simply click on "Desktop" and then click "OK".

Another popular place to install it is in your home directory. On the screenshot shown, the home directory is called "dko", because that is the username of the current user using the computer when these screenshots were taken. On your computer, your home directory name will be your username.

Once you have installed PortablePython, you can navigate to that directory using your file browser. Inside the directory, you will see several executables, as shown in Figure 1.4, “ PortablePython Install Contents”.

Figure 1.4.  PortablePython Install Contents


The installed contents are:

  • IDLE : A Python development environment.
  • IPython : An interactive Python console.
  • PyScripter : A Python development environment.
  • Python : A Python console.
  • QtDesigner : A graphical user interface designer, for designing window based applications.
