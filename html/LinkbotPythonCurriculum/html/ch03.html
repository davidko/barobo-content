Moving a Robot's Motors to Absolute Positions
This chapter will help you start moving your Linkbot around. As you go through this section, you will see small snippets of code that moves the robot. If you choose, there is a link below each snippet that allows you to see the whole piece of Python code being used to control the robot. If you choose to see the unabridged code, you may see some code that you do not yet understand. Don't worry about that yet; We'll explain the details for that "boilerplate" code later. [1]

[1] "boilerplate" code refers to code that appears at the beginning of many programs used for setup and configuration. Typically, boilerplate does not change or changes only a little bit from program to program, though it is required for the program to run properly.
