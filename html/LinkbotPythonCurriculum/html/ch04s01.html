Starting an Interactive Python Interpreter

The first step is starting an interactive Python interpreter. First, navigate to where you installed the Python software, as detailed in the section called “ Getting and Installing the Software ”. Open the folder, and you should see something like this:

Double-click on the icon labeled "IPython-Portable" to start a Python terminal. You should see the following screen pop up:

This is your Python terminal. The Python terminal is just like the Python interpreter, except you can type in one line of code at a time and it executes each line of code as you type it in.
