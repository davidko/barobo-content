Now that you are familiar with the move() function, let's try using it to move the robot around. Attach wheels to your Linkbot‑I if you haven't already and place it on a suitable surface for driving around. Now, try modifying the code below and see what you can make the robot do! 

#!/usr/bin/env python
# File: ch4ex3.py

import barobo
dongle = barobo.Dongle()
dongle.connect()

# Change the robot ID shown below to the ID of your Linkbot.
robot = dongle.getLinkbot('ABCD')
robot.setJointSpeeds(90, 90, 90)
    
robot.move( , 0, )
