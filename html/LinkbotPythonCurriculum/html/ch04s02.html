Python Variables

Variables are elements in a computer language that are used to store and manipulate data. In most languages, variables can be used to store numbers, text strings, and other types of complex data. In Python, to create [3] variables, you simply come up with a name for your variable and give it a value. Variable names can be just about anything but they have to start with a letter or the underscore "_" character, and they can't have some special symbols in the name, such as plus signs or minus signs. Otherwise, they can contain letters, numbers, underscores. There is also a list of so-called "reserved keywords" that you cannot use as variable names because they have special meaning to the Python interpreter. The reserved keywords are: 

and      assert   break    class    continue def      del      elif   
else     except   exec     finally  for      from     global   if 
import   in       is       lambda   not      or       pass     print 
raise    return   try      while    yield

Without further ado, lets try creating a variable. Type the text "myVar = 5" into your interpreter terminal and press "Enter". Your terminal should now look like this: 

In [1]: myVar = 5

In [2]:

Now type "myOtherVar = 3": 

In [2]: myOtherVar = 3

In [3]:

Now try "myLastVar = myVar + myOtherVar": 

In [3]: myLastVar = myVar + myOtherVar

In [4]:

And finally, "print(myLastVar)": 

In [4]: print(myLastVar)
8

In [5]:

What happened here? Here's what you just did:

  • First, you created a variable called myVar and gave it the value "5".
  • Next, you created a variable myOtherVar and gave it the value "3".
  • Then, you created a variable called myLastVar. Its value is the result of the calculation myVar + myOtherVar
  • Finally, with the print() function, you printed the value of myLastVar, which is 8.

As you can see, you can use variables to store numbers and also to calculate values. Now, lets try another slightly more interesting example: 

In [1]: x = 5

In [2]: x = x + 3

In [3]: print(x)
8

In [4]:

Check out line 2. If you've taken Algebra, this line might look very weird to you. With code and calculations like this, it is much better to think of the equal sign as an assignment operator rather than equality, which you may be used to from math. In other words, Python interprets this line as "Calculate the value of x + 3. Take the result and assign it to the variable x".


[3] In Programmer-speak, creating variables is often called "Declaring" variables.
