Lesson 3

Objective

Getting the Linkbot to make a right turn should be a snap. Get the Linkbot to turn right and pivot on the left wheel. Test how accurate your turns are by setting the wheels to a home position by pressing and holding the A and B buttons at the same time for a few seconds. Then running your code. 

#!/usr/bin/env python
# File: ch3lesson3.py

import linkbot

# Change the robot ID shown below to the ID of your Linkbot.
robot = linkbot.Linkbot('')

robot.setJointSpeeds(90, 90, 90)
robot.move(190, 0, 0)

Explore

Run the Linkbot on different surfaces at different speeds. Find the best combinations of speed and Degrees of rotation to accurately make a turn of 90 degrees.
