Lesson 1

Objective

In this lesson the Linkbot learns to move forward. By changing the code you can make the Linkbot move a set distance.

#!/usr/bin/env python
# File: ch3lesson1.py

import linkbot

# Change the robot ID shown below to the ID of your Linkbot.
robot = linkbot.Linkbot('')

robot.setJointSpeeds(90, 0, 90)
robot.move(360, 0, -360)
