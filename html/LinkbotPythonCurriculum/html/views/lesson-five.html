Lesson 5

Objective

Getting the Linkbot to make an about face and turn around 180 degrees is a useful maneuver. Make the Linkbot to pivot 180 degrees in one place. Try the maneuver at different speeds to find the best values to make the most accurate turn. 

#!/usr/bin/env python
# File: ch3lesson5.py

import linkbot

# Change the robot ID shown below to the ID of your Linkbot.
robot = linkbot.Linkbot('')

robot.setJointSpeeds(90, 0, 90)
robot.move(190, 0, 190)

Explore

Putting the moves together, have the Linkbot back up then turn around 180 degrees.
