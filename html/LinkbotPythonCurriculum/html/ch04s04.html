Moving your Linkbot with the Python Interpreter

Now, lets try moving the Linkbot using the interactive Python interpreter. Type the following so that your terminal looks like the following screen. If you mistype a line, don't worry; just type it again. Also, remember to replace ABCD on line 4 with the serial ID of your robot. 

 
In [1]: import barobo

In [2]: dongle = barobo.Dongle()

In [3]: dongle.connect()

In [4]: myLinkbot = dongle.getLinkbot('ABCD')

In [5]: myLinkbot.move(90, 90, 90)

In [6]: myLinkbot.move(90, 0, -90)

In [7]:

What did the robot do when you typed this code in? Lines 5 and 6 should look familiar to you from the previous chapter, "Linkbot Driver's Ed".
