Using the Python Interpreter as a Calculator

Python supports all of the math operations you might be used to on your calculator, although some of the symbols they use might be different than what you are used to on pencil and paper. Here are some common math operations you can use in Python:

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • ** : Exponent e.g. 5**2 is "five squared"
  • ( ... ) : Precedence. e.g. (4+3)*2

Note that you can just type the calculation right on the interpreter terminal and it will automatically show you the result. 

In [4]: ((5+4)*4)**3
Out[4]: 46656

In [5]:

Note that this is a special feature of the Python interpreter. When you start writing Python programs in files, you will either need to store the result in a variable or print it out with the print() function.
