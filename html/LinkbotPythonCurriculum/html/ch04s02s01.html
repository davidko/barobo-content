Getting Started with PyScripter

PyScripter is a type of program called an "Integrated Development Environment (IDE)". Typically, IDE's are capable of editing code, displaying code in a visually appealing manner, executing code, and debugging code all in one program.

You can start PyScripter by double-clicking on the PyScripter in your PortablePython installation folder.
