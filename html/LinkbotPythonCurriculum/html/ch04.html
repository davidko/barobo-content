So, what about that Boilerplate Code?

So far, up to this point, you have been relying on BaroboBrowser to run code snippet examples for you. Now it's time to graduate you to writing your own complete Python programs that can run on any computer.[2] When you write Python programs, they should be saved as a "plain-text" file. This is the same format that Microsoft Notepad and other text editors use to save text files. The reason these files are called "plain-text" is because they are unable to store information about fonts, font sizes, page sizes, and other details that other formats, such as Microsoft Word, can store.

When you execute a Python program stored in a plain-text file, the Python interpreter scans the file, line by line, and executes the instructions written in the file. To familiarize you with this process, lets begin by playing with the Python interpreter.
