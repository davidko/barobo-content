Lesson 7

Changing the LED intensity using the wheel angle as an input device.

Objective

In this example the wheel is used as an input to change the intensity of the LED. To manually set the joint angle to zero before you start, press the "A" and "B" buttons at the same time until the LED flashes. Then run the program to see what happens. Continue changing the joint one angle to see the LED change intensity. 

import barobo
dongle = barobo.Dongle()
dongle.connect()
robot = dongle.getLinkbot('9418') #enter the linkbot ID
robot.setLEDColor(255,0,0)#sets the led to red
for i in range(1,4):  #gets current joint angles and prints the result
        print ("Joint {0} angle: {1}".format(i, robot.getJointAngle(i)))
joint_one=robot.getJointAngle(1)#gets joint_one value and stores it
print(joint_one)  #prints joint_one value
intensity=abs(int(joint_one*255/180))  #changes joint_one to a range of values 0 to 255
print(intensity)  #prints the intensity value
robot.setLEDColor(intensity,0,0)  #sets LED to intensity level of current joint_one angle

Explore

Change the values to make the LED green and change intensity when the wheel is turned.
