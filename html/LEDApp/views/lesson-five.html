Lesson 5

Getting the LED to change color based on an input, like pressing the "A" button.

Objective

In this lesson you will learn how to make the "A" button an input device that will signal the Linkbot's LED to change color from red to green.

import barobo
import time
import random
dongle = barobo.Dongle()
dongle.connect()
robotID = input('Enter robot ID: ') #prompt to enter the Linkbot ID
robot = dongle.getLinkbot(robotID) #assigns “robot” as Linkbot ID number entered.
robot.setLEDColor(255,0,0)  #set the LED to red on “robot”.
def callback(mask, buttons, data): # defines a function “callback”
    if buttons & 0x02:
        robot.setLEDColor(0,255,0)
robot.enableButtonCallback(callback)

Make the Linkbot beep when the "A" button is depressed.
