Lesson 2 - Blinking the LED at different rates.

Objective

In this section you will learn how to make the LED blink on and off. You have probably noticed that we see blinking lights in several places. This section demonstrates the computer code that causes the lights to blink.

#!/usr/bin/env python3

# Filename: Lesson02.py

import linkbot
import time

robot = linkbot.Linkbot('')

for i in range (1,10):
robot.setLedColor(255,0,0)
time.sleep(.25)
robot.setLedColor(0,0,0)
time.sleep(.25)
